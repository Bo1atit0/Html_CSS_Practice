HTML Notes

HTML:HyperText Markup Language

HyperText: Text that contains links to other text that the user can easily access, It's a way of connecting different pieces of information together

Markup Language: adding instructions or tags to plain text to tell the computer how to display the text Eg: In HTML we use tags like (<p>) to indicate paragraphs and (<h1>) to indicate headings. This tells the computer to display the specified test in a paragraph or as a heading

HTML(HyperText Markup Language): is the standard way we make and design webpages. It's like the blueprint or skeleton of a webpage, giving it its basic structure and layout. Think of it as the foundation for displaying content on the web.

Metadata in HTML

Metadata is data that describes data

HTML uses the <meta> elemnt to add metadata to our HTML

<meta> elements include the 'name' and 'content' attributes.

name:what type of information is stored in the metadata

content:The actual content in the metadata

They can be used to include the author of the page and to give a description of the page
