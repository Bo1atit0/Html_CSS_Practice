HTML Notes

HTML:HyperText Markup Language

HyperText: Text that contains links to other text that the user can easily access, It's a way of connecting different pieces of information together

Markup Language: adding instructions or tags to plain text to tell the computer how to display the text Eg: In HTML we use tags like (<p>) to indicate paragraphs and (<h1>) to indicate headings. This tells the computer to display the specified test in a paragraph or as a heading

HTML(HyperText Markup Language): is the standard way we make and design webpages. It's like the blueprint or skeleton of a webpage, giving it its basic structure and layout. Think of it as the foundation for displaying content on the web.

Metadata in HTML

Metadata is data that describes data

HTML uses the <meta> elemnt to add metadata to our HTML

<meta> elements include the 'name' and 'content' attributes.

name:what type of information is stored in the metadata

content:The actual content in the metadata

They can be used to include the author of the page and to give a description of the page

Other types of metadata

To make my webpage look better when it is shared on platforms like Facebook or Twitter, I use metadata to add some basic information

Hyperlinks

Hyperlinks allow us link documents to other documents or resources, when a hyperlink is clicked it takes you to another webpage

A link is created by wrapping text or content inside the anchor <a> element with a 'href' attribute containing the webpage you want to text to link to

You can make almost anything a link,the heading of your website, an Image, you just have to wrap it in the anchor element

A title attribute can also be added to the anchor <a> elemnt. This title elemnt contains additional information about the link

NOTE:URL: Uniform Resource Locator - is a string of text that defines where something is located on the web

Document Fragments

Hyperlinks can also be used to direct users to a specific part of an HTML Document known as 'Document Fragments' rather than linking to the top of the webpage

To do this you assign an 'id' attribute to the element you want to link to

You can link to a specifiic part of another webpage or the same webpage
